Welcome to Dev Setup — a collection of modular shell scripts and dotfiles that turns a fresh Mac into a fully configured development machine. Whether you're spinning up a brand-new laptop or rebuilding after a wipe, Dev Setup handles the tedious parts so you can focus on writing code. Pick only the modules you need, run a single command, and restart to find everything in place.

## Get running in three steps

Follow this quick path from a bare Mac to a working development environment.

<Steps>
  <Step title="Clone the repository">
    Pull Dev Setup onto your machine so all scripts are available locally.

    ```bash
    git clone https://github.com/donnemartin/dev-setup.git && cd dev-setup
    ```
  </Step>
  <Step title="Prepare macOS">
    Run `osxprep.sh` to install all pending macOS updates and the Xcode Command Line Tools. If a restart is required, run the script again after rebooting.

    ```bash
    ./osxprep.sh
    ```
  </Step>
  <Step title="Run the .dots launcher">
    Pass the modules you want to the `.dots` script. Use `all` to install everything, or name individual scripts to install only what you need.

    ```bash
    # Install everything
    ./.dots all

    # Install only core tools + Python data stack
    ./.dots bootstrap brew osx pydata
    ```
  </Step>
</Steps>
